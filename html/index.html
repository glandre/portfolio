A Few Accomplishments

I have been crafting software for a decade, and, I've worked with different technologies.
Here are a few projects that recently provided me the joy of craftsmanship.

SPA for Corporative Video Sharing

In 2018 I took part of a team that built, deployed and maintained a Rich and Responsive React based Single Page Application connected with a Node/Express backend integrated with a number of APIs like AWS API Gateway and Cognito, Google Drive and Google Picker, and Zoho Payments API.

Web Platform for Jewerly Stores

My team implemented this web application in 2017. We used React JS, Redux, Redux Saga, Node JS, Express Framework, Google Cloud Datastore. Integrated with Shopify and indexed with Algolia Search API.

Co-founded a chess club

Prior to 2013, the chess scene in Campo Grande , was dull. To pursue our beloved "sport", I and my friends founded the CSX . When I was the acting president, our most notable accomplishment was the reactivation of the state's chess federation. Campo Grande's chess scene never faded!
